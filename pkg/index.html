Instructions

You can move around the 3d scene and select objects to see their matrices. Select one before starting

  1. Select any object and go to the position tab and each set each value with a number between 1 and 10. Where do you see changes in the matrix. You may notice changes in the first 3 items of the 4th column. Next, you should edit the corresponding values in the matrix and see how it effects the selected object in the scene.
  2. Select a 3d object with a identifiable texture and go to the scale tab and experiment with small numbers between -4 and +4. Notice where you see changes in the transformation matrix. You may notice changes in the diagonals: row 1 column 1, row 2 column 2, and row 3 column 3. Next select an object and zoom in closely to it. set the scale of 0.001 and notice what happens. You should see the shape squish to create a flat shape. This value is use in place for 0 because 0 removes all of the color from the object. Also set the scale to -1, notice how the textures on the object flip, almost like the shape is a mirror image
  3. Select a new object and go to the rotation tab and start incrementing values slowly by 0.1. You should see the object rotate on the scene and see specific matrix values change. For rotations any numbers in the top 3 columns and rows can change. You can try to edit these values in the matrix afterwards
  4. Select any object and experiment with some of the other values in the matrix before clicking the "Advanced" tab. Besides rotation, scaling, and position, you might also notice the shape stretch diagonally taking on a rhombus shape. This transformation is called a shear.
  5. next click the advanced tab.
  6. Select any object and go to the 4th row and column. Edit this value with small numbers. You should see the object shrink and move closer to the center with values greater than 1. You should also see the object get bigger and further away from the center when this value is greater than 1. Notice how changing this value does not modify any information under the scale/position tab where it clearly should. This was a design choice by the 3js library to make their code simpler because this value is seldomly edited.
  7. Go to the 4th row and one of the first 3 columns. Edit one of these numbers to be a very small value such as 0.05. You should see the object stretch a little bit in one direction. Then set it to a big value, you should see a scary sight of the shape stretching out towards infinity. These transformations are not really used to modify shapes as much as they are used for projection or making the scene look 3d.

Troubleshooting

I cannot click on a shape.

- shapes with scales set exactly to 0 cannot be clicked. Also shapes that have anything in the fourth row edited may be hard to click

- solution: Try reloading the page.

The components are too small for my screen / The GUI is lagging

- This application is mainly for Ipads and computers.

- Solution: Try switching to a desktop device.

The scene comes up blank or will not load.

- This application is not supported by IE or Opera and may not be supported by old browsers.

- Solution: Try switching to a new browser or update your existing one.
